# 系统整体架构说明

```text
命题方原始数据（服创数据集/）
    → 预处理（src/preprocess/*，NetCDF → npz/统计量）
    → data/processed/{hydro,eddy,anomaly,stats}
    → 算法层（src/hydro, src/eddy, src/anomaly）
    → outputs/{hydro,eddy,anomaly}（权重与指标）
    → 演示 / 提交（src/demo, submission/）
```

## 分层说明

| 层 | 内容 |
|----|------|
| **配置** | `config/data.yaml`（数据根、划分、`hydro_preprocess`）；`config/hydro_hycom.yaml`（水文网格与步长）；`config/variable_map.yaml`（NetCDF 变量名别名） |
| **数据** | 原始 `服创数据集/`（默认 gitignore）；产物 `data/processed/hydro/` 等，见 `data/README_data.md` |
| **水文** | 预处理 `hydro_dataset` / `hydro_nc_stack`；训练 `src/hydro/train.py`；评测 `src/hydro/eval.py` |
| **工具** | `src/utils/config.py`（路径解析）；`src/preprocess/netcdf_io.py`（中文路径等兼容） |
| **脚本** | `scripts/setup_fuchuang_to_data.sh`（数据部署/软链）；`scripts/run_*.sh`（按需） |

详细接口与验收项仍以仓库根《A09-项目开发文档》为准（若本地仅 `相关文件/` 副本，请与团队主仓同步）。
