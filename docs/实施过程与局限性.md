## 水文管线已知局限

1. **陆地与缺测**：HYCOM NetCDF 常见 `short` + `_FillValue`（如 −30000），陆地格点为缺测；预处理中会对非有限值计数并 **`nan_to_num`→0**，Z-score 使用 **`nanmean`/`nanstd`** 与输出兜底。物理上陆地格点参与 MSE 会拉偏指标，后续可改为**海掩膜损失**（未在基线实现）。
2. **滑窗内存**：`window_stride` 过小（如 1）时窗口数 N 极大，**内存可达百 GB 级**；`config/data.yaml` 默认 `window_stride: 24`，`build_windows` 会打印预计 GiB 与填充进度。
3. **显存**：长序列训练已采用合并解码头、`encoder_checkpoint`、小 batch、梯度累积等；仍 OOM 时需降 `hidden_dim`/`num_layers` 或关 AMP。
4. **划分**：开发期可用 `split` 比例；正式评测应启用 `hydro_year_split` 与 `--year-split` 预处理，与命题方年份一致。
5. **`data/processed/anomaly`、`eddy` 为空**：仅跑水文时属正常，见 `data/README_data.md`。
