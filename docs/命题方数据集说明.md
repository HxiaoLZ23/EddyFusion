# 命题方数据集说明（摘要）

> **原始全文**见仓库内 `服创数据集/数据集说明.md`（参赛方提供，更新日期以该文件为准）。  
> 本文档便于在无数据目录的机器上查阅；**评测划分与指标以命题方为准**。

## 通用注意事项

1. **不要求**部署到云服务器。  
2. **禁止**将验证集用于训练。  
3. 各子任务海区、分辨率、变量与划分如下。

---

## 一、中尺度涡识别

| 项目 | 说明 |
|------|------|
| 海区 | 黑潮延伸体 **25–45°N，140–180°E** |
| 分辨率 | 空间 **0.125°**；时间 **daily** |
| 数据来源 | META4.0 DT 涡旋轮廓标注等（见命题方参考链接） |
| 变量 | **ADT**（海面动力高度）；**UGOS**（地转流 u）；**VGOS**（地转流 v，注意与命题方英文描述核对） |
| 指标 | **IoU**；气旋涡与反气旋涡为**两类**目标 |
| 划分 | 训练 **1993-01-01～2022-12-31**（对应 nc 如 `19930101_20021231.nc` 等）；测试 **2023**；验证 **2024**（见原始表） |
| 备注 | 二维变量可作**多通道图像**输入，无需再绘图 |

---

## 二、海域要素预测（水文）

| 项目 | 说明 |
|------|------|
| 海区 | **黄渤海 31–41°N，117–127°E** |
| 分辨率 | 空间 **0.08°**；时间 **hourly** |
| 数据来源 | GOFS 3.1 HYCOM 再分析等（见命题方链接） |
| 变量 | **SST、SSS、SSU、SSV**（**不可自行增加变量**；扩展思路可写在扩展方案中） |
| 指标 | 命题方「均方误差」指 **NRMSE**（归一化 RMSE 口径） |
| 输入输出 | 输入：过去 **n** 小时与当前时刻 **[t−n, …, t−1, t]**；输出：未来 **72 小时 [t+1, …, t+72]**（小时级步长则输出 **72** 个时间步） |
| 划分 | 训练 **1994～2013**（按年目录）；测试 **2014**；验证 **2015** |

## 三、风浪异常识别

| 项目 | 说明 |
|------|------|
| 海区 | **西太平洋 0–60°N，100–180°E** |
| 分辨率 | 风场 **0.25°**；波浪场 **0.5°**；时间 **3-hourly** |
| 变量 | **U10、V10、MWD、MWP、SWH** |
| 指标 | **准确率**：真实发生的台风中**被正确识别**的比例（开放题设，重思路创新） |
| 划分 | 训练 **2014～2023**；测试 **2024**；验证 **2025** |
